# quecto

> **The smallest harness of all time.**

`quecto` — the [SI metric prefix](https://en.wikipedia.org/wiki/Metric_prefix) for **10⁻³⁰**, the smallest unit in the metric system.

If *kilo* is 10³⁰ and *quecto* is 10⁻³⁰, this project lives at both extremes: a universal harness built from the smallest possible composable units.

## What is quecto?

The smallest AI harness you'll ever find.

One job: take a prompt, run it through an LLM, return the output.

Two small functions. Two files. That's the whole harness.

Build everything else on top.

## Use it for

- **Agents** — multi-step reasoning, tool use, autonomous workflows
- **Data** — extract, transform, classify, summarize at scale
- **Code** — scaffolding, refactoring, reviews, tests
- **Content** — writing, editing, SEO, translation, formatting
- **Research** — fact-checking, synthesis, comparison, deep-dive
- **Anything** — if an LLM can reason through it

## Why the name?

```
… → mega (10⁶) → kilo (10³) → base → milli (10⁻³) → micro (10⁻⁶) → … → quecto (10⁻³⁰)
```

`quecto` is the smallest possible unit. This project takes that literally: break any task down to its smallest composable piece, then compose them back up.

## Status

**Early / experimental.** This is a pet project — building in the open.

## Philosophy

1. **LLMs are the backend.** The harness is just the glue.
2. **Everything is composable.** Small pieces → big things.
3. **Describe it, run it.** If you can explain it to an LLM, quecto handles it.
