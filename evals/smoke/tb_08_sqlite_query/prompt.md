The SQLite database `store.db` has a `products` table. Write a Python script `query.py` that:
1. Opens `store.db` using the `sqlite3` standard library.
2. Finds the product with the highest `price`.
3. Writes its `name` and `price` to `result.txt` in the format: `name: <name>, price: <price>`.
Then run `query.py` to produce `result.txt`.
